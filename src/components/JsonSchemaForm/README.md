
# JsonSchemaForm Component Contract

- it has Props of:

  schema: ISchemaType - JSON for the form;
  uiSchema: Object - UI properties;
  formData?: Object - Initial values of form;
  widgets?: Object - custom widgets;

- it has no State

- it always renders a

-  `<div>` element,

-  `<form>` element,

-  `<button>` element,
